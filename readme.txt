== Description ==
The Performance Scoring Embed plugin allows you to embed custom survey forms generated by our Performance Scoring app into your WordPress site. Use our shortcode [performance_scoring formid="2"] or add the Gutenberg block to display your survey.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/performance-scoring-embed` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings -> Performance Scoring to configure your global settings.

== Frequently Asked Questions ==
= How do I embed a survey? =
Use the shortcode [performance_scoring formid="your_form_id"] or add the Gutenberg block.
